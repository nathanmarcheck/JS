funções em JavaScript

As funções são subrotinas que deixam o código modular mais legivel fazendo com que o código possa ser reotilizado. uma função realiza algo específico e pode ser requisitado a qualquer momento.

Exercícios

  1. Crie uma função que leia 3 números digitados pelo usúario. E realize a multiplicação dos valores e apresnete na tela. A função deve ser com parametro e não precisa ter retorno. solução
  2. altere o exercício anterios em que haja uma função para ler os valores digitados pelo usuário e uma função para calcular a multiplicação e ainda uma ultima função para exibir o resultado. solução
  3. crie um script em que o usuário digite a nota da prova do trabalho e a nota da lista de exercícios e apresente se ele foi aprovado, reprovado ou exame. A média para a aprovação é 7. caso a media fique entre 4 e 6.9 ele está de exame. caso contrário ele está reprovado.utilize funções com parametros e com retorno. para o caulculo da média ponderada considere:70% para prova; 20% para o trabalho e 10% para a lista de exercícios. solução
  4. faça um script que realize a conversão de graus celsius para fahrenheit. solução
