aprendendo DOM

  1. getElementBy("nome") - pegar elementos pelo id
  2. innerHTML - inserir ou mudara conteúdo
  3. mudar estilo
  4. classList - insere uma classe ao elemento html
    • .add - adiciona uma classe ao elemento.
    • .remove - remove uma class do elemento
    • .taggle - insere ou remove a class se tiver o elemento
  5. createElement - cria elemntos.
  6. querySelectorAll
